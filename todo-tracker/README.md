# To-Do List

A to-do list app built with plain HTML, CSS, and JavaScript (no frameworks, no build step).

[Live demo](https://ewicaondrasakova.github.io/learning-projects/todo-tracker/html/index.html)

## Features

- Add tasks with priority, tags, and notes
- Recurring tasks (daily, weekly, or custom days)
- Calendar view showing completed days
- Search and filter tasks
- Progress rings, weekly/monthly success rate, and a 14-day activity heatmap
- Simple profile with nickname, avatar, and optional email
- Slovak / English language toggle

## Run locally

Just open `html/index.html` in a browser — no build or install needed.
